# Spam Email Classifier

This project classifies emails into Spam and Ham using Machine Learning.

## Features

- Text Preprocessing
- TF-IDF Vectorization
- Random Forest Classifier
- Streamlit Web App
- Custom Email Prediction

## Tech Stack

- Python
- Scikit-learn
- Streamlit
- NLTK
- Pandas

## Run

pip install -r requirements.txt

streamlit run app.py
